# 随机数生成器

特点：

1. 随机生成设定范围内的整数
2. 支持自定义数据库，随机数可以从数据库中获取
3. 支持 window、mac 和 Linux 平台

# 相关技术

1. 跨平台使用 electron 技术
2. 前端使用 react 框架，ui 库使用 antd
3. 使用 vite 打包
